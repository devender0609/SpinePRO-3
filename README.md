# SpinePRO Joint-CAT (PROMIS + SRS) — Prototype

Static (no backend) Vercel-ready deployment.

## Routes
- `/` -> `index.html`
- `/survey` -> `survey_joint.html`
- `/results` -> `results_joint.html`

## Notes
- Item parameters are estimated from a synthetic dataset sample and are **not** official PROMIS or SRS calibrations.
- Domain summaries on the results page are **prototype proxies** for clinician interpretability until you finalize IRT calibration and validation.
